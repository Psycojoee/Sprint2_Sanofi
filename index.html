Aplicativo Sanofi

A empresa Sanofi dispobiliza um aplicativo para os seus colaboradores, porém muitas função do mesmo ainda não estão totalmente funcionais. Como por exemplo, o pessoal que utiliza o sistema Android, não consegue realizar o download do mesmo, sendo assim, não possuem acesso. Algumas páginas do aplicativo, simplesmente ficam em branco quando o usuário tenta acessar.

Principais Erros Encontrados

  • Menu Não Corresponde

    Podemos observar que nessa página o menu "TRATAMENTO" acaba não levando a lugar algum, sendo assim o usuário não consegue prosseguir. Ajustaremos o botão para que o usuário consiga utilizar o aplicativo sem nenhuma interrupção.

  • Telas Em Branco

    Muitas opções dentro do aplicativo acabam gerando páginas em branco onde os usuários não conseguem absorver a informação. Resolveremos as páginas em branco com as aplicações corretas.

  • Menu Pouco Intuitivo

    Podemos observar na foto que o menu do aplicativo não é tão intuitivo, sendo assim criando uma grande dificuldade para os usuários. Criaremos um menu muito mais intuitivo com a utilização de chatbots para auxiliar caso o usuário ainda tenha dificuldades.

  • Versão Android

    Quando um usuário Android tenta localizar o aplicativo em sua loja, por estar desatualizado ele não aparece. Nossa solução é atualizar o aplicativo e disponibiliza-lo na loja para download.
